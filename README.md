# App
"Mitigating Different Malware Attacks"

The main goal of this research is to create and implement a software application that can detect and reduce various types of malwares on user devices like desktop, laptops, and personal computers.

Technologies Used:Python

Installation Guid

1. Install Python, go to https://python.org
2. Clone the Repository https://github.com/Via-321/App.git
3. Open VSCODE
4. Open this Folder
5. Go to terminal and type pip install -r requirements.txt
6. Then run python app.py
